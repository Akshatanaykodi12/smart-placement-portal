# Smart Placement Portal Frontend

This is the React frontend for the Smart Placement Portal, built with Vite.

## Setup

1. Install dependencies:

```bash
cd frontend
npm install
```

2. Start the development server:

```bash
npm run dev
```

The frontend will run on `http://localhost:5173` and connect to the backend at `http://localhost:5000`.

## Features

- Student registration and login
- Student dashboard with profile and resume upload
- Job eligibility filtering
- Admin analytics dashboard

## Components

- `Login`: Student login form
- `Register`: Student registration form
- `Dashboard`: Student profile and resume upload
- `JobList`: List of eligible jobs for the logged-in student
- `AdminAnalytics`: Admin dashboard with analytics
